Game Maker - Quick Guide

Definizioni

Oggetto visible
indica se le istanze di questo oggetto sono visibili
Oggetto solid
le collisioni vengono trattate in modo differente. usare solid solo con i muri
Create Event
settare le variabili iniziali
destroy Event
azioni da eseguire prima della sua distruzione
Alarm Event
cambi di direzione, ogni tot andature
Collision Event
oggetto solido
End Of Animation
usato soprattutto per le esplosioni, da eseguire una volta sola
Move Torwards a Point
seguire un'altra istanza

Cambiare orientamento / direzione oggetto

  1. Evento CREATE -> set variable "direction" to ??
  2. Si può fare in due modi:
    1. Evento (di solito è un tasto) -> set variable "direction" relative to ??
    2. Evento STEP -> set variable direction relative to ?? (cambio automatico di direzione).
  3. Trasformare lo sprite: Evento END STEP -> transform sprite (angle = direction).

Se l'oggetto deve muoversi (alla pressione di un tasto o ad ogni step) si crea l'azione Move con direzione "direction".
